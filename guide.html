Setup Guide

Get started in 4 simple steps ‚Äî takes less than 2 minutes

1

Download the Extension

Click the "Download Extension" button above to download the NetMirror-Extension.zip file to your computer.

Save it somewhere easy to find, like your Desktop or Downloads folder.
2

Extract the ZIP File

Right-click the downloaded .zip file and select "Extract All" (Windows) or double-click to unzip (Mac). You'll get a folder called extension.

Don't delete this folder ‚Äî the browser needs it to run the extension.
3

Load in Your Browser

Open your browser's extension page:

Microsoft Edge: Go to edge://extensions
Google Chrome: Go to chrome://extensions

Then:

1. Enable "Developer mode" (toggle in top-right corner)
2. Click "Load unpacked"
3. Select the extracted folder (the extension folder)

Make sure you select the folder that contains manifest.json, not the outer folder.
4

Launch Netflix üéâ

Click the NetMirror icon in your browser toolbar to open the extension popup. It will automatically connect to the server.

Click "Launch Netflix" and you'll be signed in automatically. Enjoy streaming!

Pin the extension to your toolbar for easy access ‚Äî click the puzzle icon ‚Üí pin NetMirror.
